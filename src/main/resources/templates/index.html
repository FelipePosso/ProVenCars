ProVenCarz

Este proyecto formativo para la realización de un sistema de información web cuenta con funcionalidades como:
1. Consultar todos los registros de los vehiculos registrados o existentes en el sistema
2. Filtrar la consulta de los vehiculos registrados o existentes en el sistema por Categoria
3. Filtrar la consulta de los vehiculos registrados o existentes en el sistema por un Rango Mínimo y Máximo de Precio
4. Agregar un nuevo vehículo al sistema
5. Editar los datos de un vendedor como nombre, apellido, correo y número
